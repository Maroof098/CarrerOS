# CareerOS

CareerOS is an AI-powered career management platform designed for engineering students.

## Features

- Personalized Career Roadmaps
- Daily Task Management
- AI Mentorship
- Project Planning
- Resume Builder
- Progress Tracking
- Dark/Light Theme Support
- Firebase Authentication

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- Firebase Firestore

### AI
- Google Gemini API

## Goal

CareerOS helps students organize their learning journey, track progress, build projects, improve resumes, and prepare for placements through AI-powered guidance.
